② 左侧参数面板详解
控制整个系统核心计算逻辑的参数区
项目信息
项目名称 用于保存文件时的命名，显示在标题栏
机组台数 影响单台报价计算，总价 = 单台价 × 台数
有/无开关 控制水轮机/发电机/阀门是否计入报价
加价系数
自制加价 % 自制件成本 × (1 + 自制加价率) = 报价
默认 100%，即成本翻倍
外购加价 % 外购件成本 × (1 + 外购加价率) = 报价
默认 10%，外购件利润较低
运输参数
运输距离 km 工厂到项目现场的公路距离
普通件单价 元/吨·公里，适用于常规设备
超宽件单价 需要特殊车辆运输的大型设备单价
超宽件重量 填 0 则全部按普通件计算运费
重量附加系数 含包装箱附加，通常取 1.05
🔄 绿色字段为自动汇总值（从材料明细表计算），双击可手动覆盖，再次双击恢复自动值

⑤ 各功能模块详解
顶部导航栏各 Tab 页面的用途说明
📊 概览 — 投标总价汇总与分析
顶部 5 个 KPI 卡片：拟投标总价、单台报价、固定成本、消化占比、代理服务费
报价构成饼图：各分项占总价的比例，直观看出成本结构
吨价对比图：材料吨价 vs 投标吨价，判断报价是否合理
报价明细清单：逐行展示各设备的自制/外购成本、加价后报价及占比进度条
⚠ 消化占比 > 45% 时，系统会显示红色预警提示
⚙️ 三大主机 — 水轮机/发电机/阀门成本卡片
每台设备显示：重量、材料价、外购件价、自制部分、材料吨价、拟报单价、投标吨价
自制 vs 外购占比条形图，颜色区分蓝色（自制）和琥珀色（外购）
点击重量旁边的 → 按钮可跳转到对应设备的材料明细表
📋 材料明细 — BOM 成本录入（最核心模块）
分为：水轮机 / 发电机 / 进水阀(参考) / 阀门整机 四个子标签
采用树状层级结构，序号如 "一→1→1.1→1.1.1" 表示层级关系
蓝色汇总行（序号为"一"或纯数字的行）自动汇总子行数据，不可手动编辑金额
材料选择：单击"材料"列，弹出材料库选择器，支持分类浏览和搜索
利用率：取自材料明细行；金额 = 重量 ÷ 利用率 × 材料库标准价格，自动计算
自制/外购：点击"类别"列可切换，外购件会被单独汇总到外购成本
底部"修复汇总"按钮可修正因手动编辑导致的汇总数据不一致问题
🔧 备件工具 — 备品备件与专用工具清单
分水机/发机/阀门三个子标签
数量列只显示纯数字，单位列单独显示（套/台套/把等）
备件表：记录各备件名称、规格、数量单位、自制金额、外购金额
工具表：额外有重量字段，影响运输费计算
阀门备件/工具金额单位为元（其余为万元），汇总时自动换算
🤖 自动化/监测 — 元件清单与在线监测设备
主机自动化：列出各传感器、阀门、变送器等元件的型号、单价、数量、合价
在线监测：振摆监测、局放监测、气隙监测等系统设备
两张表的金额均为元，汇总时换算为万元显示
支持关键字搜索（元件名称/型号/制造厂）
🚛 运输费 — 自动计算运输成本
根据左侧面板的运输参数自动计算，显示详细的运费拆解：
总重量 → 附加后实际运输重量 → 超宽件/普通件分别计费 → 汇总运费
运费 = 普通件重量 × 普通单价 × 距离 + 超宽件重量 × 超宽单价 × 距离
🎯 竞价分析 — 报价策略与胜率预测
胜率仪表盘：基于报价/成本比例的经验模型预测中标概率
最低可报价：保本线（成本价），低于此价亏损
建议报价区间：在保本线基础上加合理利润的区间
敏感度分析：各成本项目对总价的影响权重
价格阶梯模拟：不同报价下利润和胜率的变化曲线
报价策略建议：根据当前成本结构自动生成文字建议
🗄️ 材料库 — 管理材料标准价格数据库
维护材料名称、分类、标准价格（万/T）的基础数据；利用率只保存在材料明细行中
一键导入：从 Excel 文件批量导入材料价格，支持冲突预览
批量更新：根据材料库的最新价格批量更新材料明细表中的金额
导出：将材料库导出为 Excel 文件
材料库中的数据会在"材料明细"编辑时的下拉选择中显示
⑥ 保存与导出
数据持久化和文件管理
保存方式
💾 保存按钮（推荐）
点击顶部「💾 保存」按钮，将所有表格数据和参数嵌入到 HTML 文件中下载，下次打开该文件时数据完全恢复。适合存档和分享。
🔄 自动保存
每次编辑后数据自动保存到浏览器 localStorage，关闭后重新打开同一 HTML 文件数据也会恢复（同一浏览器、同一文件路径）。
📤 导出 CSV
顶部「导出CSV」按钮可将当前材料明细数据导出为 CSV 格式，可用 Excel 打开查看。
快捷键
快捷键 功能
Ctrl + S 保存 HTML 文件（含全部数据）
Alt + 1~9 快速切换到对应编号的 Tab 页
🌙 按钮 切换暗色/亮色主题
⚠️ 注意：浏览器 localStorage 存储有限，数据量大时建议定期点击「💾 保存」生成带数据的 HTML 文件存档，不要只依赖 localStorage。
⑦ 方案对比功能
保存多个报价方案并横向对比
使用场景：当需要对同一项目制定"激进报价"、"保守报价"、"最优报价"等多套方案时，可以使用方案对比功能。

操作步骤：
① 调整好左侧参数（加价率、运输等）

④ 切换参数后继续保存第二个方案
⑤ 在「方案对比」Tab 查看横向对比表格

最多可保存 8 个方案，支持删除和加载（点击加载可恢复为该方案的参数）。

⑧ 常见问题 FAQ
使用中遇到的典型问题与解答
Q 为什么修改了材料数据，左侧参数没有变化？
左侧面板的重量/金额字段标注为"自动汇总"的（绿色边框）会随材料明细的修改实时更新。如果发现数据不一致，可点击左侧一致性警告横幅中的「一键修复所有不一致」按钮，或到材料明细底部点击「修复汇总」。
Q 材料明细中有 ⚠ 红色警告图标是什么意思？
表示该汇总行的重量或金额与其子行数据之和不一致。通常是手动修改了汇总行，或子行数据变动后未重新汇总。鼠标悬停在 ⚠ 上可查看期望值，点击「修复汇总」自动修正。
Q 阀门备件和工具的金额怎么是"元"不是"万元"？
阀门备件（valve_parts）和阀门工具（valve_tools）的金额字段原始单位为元，而水机/发机备件工具为万元。系统在汇总时会自动将元值除以 10000 换算为万元。表格下方的汇总信息会同时显示原始元值和换算后的万元值。
Q 数据丢失了，如何恢复？
如果使用了「💾 保存」下载过 HTML 文件，直接打开该文件即可恢复所有数据。如果只依赖 localStorage 但数据丢失（清除浏览器数据、换浏览器等），则无法恢复。建议定期点击「💾 保存」生成存档文件。
Q 如何重置某张表格到初始数据？
在对应表格下方的「恢复默认」按钮，可以将该表格恢复到系统内置的初始数据（那读电站原始数据）。左侧参数面板中也有「重置默认值」按钮，恢复所有参数到默认值。注意这会清除该表格所有自定义修改。
💡
使用建议：首次使用建议先在「三大主机」Tab 确认设备配置，再到「材料明细」录入各零部件数据，最后通过「概览」查看汇总结果，利用「竞价分析」确定最终投标价格，并用「方案对比」保存多套方案供决策参考。
